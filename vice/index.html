Introduction Last updated: 2023-02-15

Vice is (now) an air traffic control simulator, focused on the TRACON. Its goal is to provide a training experience similar to a VATSIM sweatbox with an instructor, just without the instructor, making it possible to better prepare for sweatboxes ahead of time. It presents a STARS-like interface, supports the same commands as vSTARS, but with simulated traffic and other controllers.

It currently only supports departure and arrival control scenarios at JFK, though adding more airports wouldn't be too much work; PHL scenarios will likely come next. See the short video overview for more information.

ATC Commands

A number of commands are available for issuing control instructions to aircraft. To use them, type the appropriate command and click on an aircraft's track; on the top of the screen you can see the (virtual) pilot's readback. The aircraft will then start following that instruction, to the best of its abilities.

If you'd like to issue multiple commands to an aircraft, enter them one after another with a space between them, and then click on the appropriate aircraft.

Command Function Example
Hheading Directs the aircraft to fly the specified heading. It will turn in whichever direction gets it to that heading most quickly. H050
Lheading Directs the aircraft to turn left to the specified heading. L180
Rheading Directs the aircraft to turn right to the specified heading. R180
Dfix Directs the aircraft to proceed direct to the given fix. (The specified fix must be in the aircraft's flight plan, including on the approach assigned to it.) DWAVEY
Calt Directs the aircraft to climb to the specified altitude, which is given in hundreds of feet. C170
Dalt Directs the aircraft to descend to the specified altitude, given in hundreds of feet. DWAVEY
Sknots Gives the aircraft a speed restriction. S250
Eapproach Tells the aircraft to expect the specified approach. This command must be used before an aircraft is cleared for an approach. It also adds the approach fixes to the end of the aircraft's flight plan H050
Eapproach Tells the aircraft to expect the specified approach. This command must be used before an aircraft is cleared for an approach. It also adds the approach fixes to the end of the aircraft's flight plan H050
X Deletes the specified aircraft from the simulation. X

Installation

Vice provides installers for both Windows and Mac that make installation and upgrades straightforward. On Linux, a more manual process is currently necessary.

Windows

To install vice on Windows, download and launch the installer below. After installation, you can find the vice application in the Program Files (x86)/Vice folder. (Apologies that the installer does not yet offer the option of adding vice to your program menu or making a desktop shortcut; that will be remedied in a future release.)

Your browser may warn about the installer being from an unknown publisher and when you run the installer, Windows will put up a window informing you that it has protected your PC and prevented the installer from running. Click "More Info" and then "Run anyway" to proceed with installation.

Download Vice v0.2.1 for Windows

Macintosh

On a Macintosh computer, download the zip file below; it contains a universal binary that runs on both Intel (x86) and Apple (M1, ...) CPUs. Note that MacOS Big Sur or a more recent version of OSX is required. After opening the zip file, drag Vice.app to your Applications folder to install it.

Download Vice v0.2.1 for Mac

Linux

On Linux systems, it is currently necessary to install vice from source. (We will provide a simpler installation process in the future.) See the directions on building vice on Linux in the source code distribution for details.

Reporting Bugs

If you encounter bugs in vice, apologies! It would be of great help if you would send in a report if vice crashes or if you see mistakes in how it simulates aircraft or the STARS interface.

The best way to report bugs is vie email at matt@pharr.org. Alternatively, if you have a github account, you can file bugs directly in vice's issue tracker.
